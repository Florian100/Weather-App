# Weather App 

A real-time weather application built with Vanilla JavaScript that fetches data from the OpenWeatherMap API.

## 🛠️ Key Features
- **Live Data Fetching:** Integrates with a REST API to get real-time weather information.
- **Dynamic Icons:** Displays weather-specific icons (sun, clouds, rain) based on the city's current conditions.
- **Interactive UI:** Smooth user experience with "Enter" key support and instant data display.
- **Responsive Design:** Works perfectly on both desktop and mobile devices.

##  Tech Stack
- HTML5 & CSS3 (Custom design)
- JavaScript (Async/Await & Fetch API)
- OpenWeatherMap API
